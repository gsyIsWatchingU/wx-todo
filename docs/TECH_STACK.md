# 技术栈

## 前端

- **Taro** v4.x - 跨平台小程序开发框架
- **React** v18.x - UI 框架
- **TypeScript** v5.x - 类型安全

## UI 组件库

- **NutUI React Taro** - 京东 NutUI React 版本，支持 Taro

## 后端服务

- **Supabase** - 开源 Firebase 替代品，提供数据库、认证、存储等功能

## 开发工具

- **Taro CLI** - Taro 命令行工具
- **微信开发者工具** - 小程序预览和调试
