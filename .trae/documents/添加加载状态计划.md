# 添加查询过程中的中间态（加载状态）

## 需求分析

用户反馈在查询任务时，一开始显示"暂无任务"，但实际上只是正在查询中，需要添加加载状态提示，提升用户体验。

## 当前问题

- `today/index.tsx` 中已有 `isLoading` 状态，但未传递给子组件
- 子组件（DayView、WeekView、MonthView、ListView）只显示空状态，无加载提示
- 用户在网络较慢时会误以为没有任务

## 实现方案

### 1. 修改 today/index.tsx
- 将 `isLoading` 状态传递给各个视图组件

### 2. 修改 DayView.tsx
- 添加 `isLoading` 属性
- 加载中显示加载动画/提示

### 3. 修改 WeekView.tsx
- 添加 `isLoading` 属性
- 加载中显示加载动画/提示

### 4. 修改 MonthView.tsx
- 添加 `isLoading` 属性
- 加载中显示加载动画/提示

### 5. 修改 ListView.tsx
- 添加 `isLoading` 属性
- 加载中显示加载动画/提示

## 修改文件列表

1. `src/pages/today/index.tsx` - 传递 isLoading 状态
2. `src/pages/today/components/DayView.tsx` - 添加加载状态显示
3. `src/pages/today/components/WeekView.tsx` - 添加加载状态显示
4. `src/pages/today/components/MonthView.tsx` - 添加加载状态显示
5. `src/pages/today/components/ListView.tsx` - 添加加载状态显示

## 风险评估

- 低风险：仅添加 UI 状态提示，不影响核心业务逻辑
- 需要确保加载状态正确传递和显示

## 测试验证

1. 进入 today 页面时应显示加载状态
2. 加载完成后显示任务列表或空状态
3. 切换视图模式时应显示加载状态
