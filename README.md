# wx-todo

微信小程序 Todo List 应用。

基于 Taro + React + TypeScript + Supabase 开发。

## 快速开始

### 安装依赖

```bash
npm install
```

### 配置环境变量

复制 `.env.example` 为 `.env`，填入 Supabase 配置：

```bash
cp .env.example .env
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览

使用微信开发者工具导入项目根目录，点击预览。
